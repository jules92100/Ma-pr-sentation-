Mon Parcours

Mon aventure dans le code n'a pas commencé dans une salle de classe, mais par besoin d'un refuge. Victime de harcèlement scolaire, j'ai trouvé dans la programmation un espace où je pouvais me reconstruire.

J'ai débuté sur Roblox Studio (Lua), apprenant la logique, les événements et la structure. Ce qui n'était qu'un moyen d'échapper à la réalité est devenu une passion technique. J'ai ensuite évolué vers Python, Java et le développement Web pour créer des outils concrets et sécurisés.

Projet Phare : Les Voix Claires 92

Association contre le harcèlement scolaire

J'ai conçu et développé le site web pour cette association. Ce projet va au-delà du code : il s'agit d'un espace de confiance.

  • Langages : HTML, CSS, JavaScript (Intégration).
  • Sécurité & Légalité : Mise en place d'une architecture conforme au RGPD.
  • Engagement : Priorité absolue à la confidentialité des données des utilisateurs et à la robustesse du site face aux accès non autorisés.
